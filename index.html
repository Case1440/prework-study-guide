HTML

  • HTML stands for Hypertext Markup Language
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Some other elements that are used to add content to a page are: Text elements, Image elements, Header elements, and List elements
  • There are also elements that aren't seen by users such as Meta-related, Content division, and Semantic. These are used in the background to divide and describe a page.
  • HTML elements work together to build the structure of a webpage and provide a framework that makes it easy for a user to navigate the webpage.

CSS

  • There are three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet.
  • Inline CSS: When a developer wants to style an element directly in the HTML file.Inline styles apply to a specific HTML tag, using a style attribute with a CSS rule to style a particular page element only. They’re useful for quick and permanent changes but are less flexible than external and internal style sheets, because each inline style must be edited individually whenever you make a design change.
  • Internal CSS Style Sheet: contains CSS rules for the webpage in the head section of the HTML file. The rules only apply to that page, and you can configure CSS classes and IDs, which are used to identify elements, to style multiple elements in the page code. Again, a single change to the CSS rule will apply to all tagged page elements.
  • External CSS: Most common way to style a webpage. Allows developers to kep all their CSS rules in a seperate file.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
